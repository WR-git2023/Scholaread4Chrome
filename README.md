# Scholaread4Chrome

**中文** | [English](README.en.md)

Chrome 浏览器插件（Manifest V3）：实时检测网页中的 PDF 资源，一键下载、一键添加到 Scholaread（靠岸学术）阅读列表，并可选择存入指定文件夹。

> ⚠️ **声明**：本插件为非官方第三方工具，与 Scholaread / 坚果云官方无任何关联。插件复用浏览器中已有的 Scholaread 网页登录会话，不需要任何账号密码或 API Key。请仅用于个人学习与科研用途。

## 📥 下载安装（2 分钟）

1. 打开 **[Releases 页面](../../releases)**，下载最新版的 `Scholaread4Chrome-vX.X.X.zip`；
2. 解压到任意目录（例如 `D:\Scholaread4Chrome`）；
3. Chrome 地址栏访问 `chrome://extensions`，打开右上角 **开发者模式**；
4. 点击 **加载已解压的扩展程序**，选择解压后的目录；
5. 在 [scholaread.cn](https://www.scholaread.cn) 登录你的账号，即可开始使用。

## ✨ 功能特性

- **PDF 实时检测**：自动识别 `.pdf` 直链、`/pdf` 路径、`citation_pdf_url` meta、内嵌 PDF、arXiv 摘要页、IEEE Xplore 摘要页等；支持 arXiv、IEEE Xplore、MDPI、Wiley、EuropePMC 等主流学术站点；
- **悬浮胶囊**：检测到 PDF 时页面右下角出现悬浮胶囊（图标 + 数量角标 +「添加」键），可拖拽，点击图标展开完整列表；
- **一键添加到 Scholaread**：论文直接加入你的阅读列表并打开阅读器；IEEE 等需要机构权限的 PDF 会自动在浏览器内下载后上传真实文件，不会出现"添加后打不开"的情况；
- **添加时可选文件夹**：下拉选择已有文件夹（默认不加入），或输入名称新建文件夹；
- **进度反馈**：悬浮胶囊圆环进度条（蓝=进行中 / 绿=成功 / 琥珀=已在库中 / 红=失败），弹窗内为条目进度条；
- **一键下载**：每条 PDF 均可直接浏览器下载；
- **登录状态同步**：弹窗显示 Scholaread 登录状态（昵称/头像/账户类型），未登录一键跳转。

## 📖 使用简版

1. 先在 [scholaread.cn](https://www.scholaread.cn) 登录；
2. 打开任意论文页面（arXiv、IEEE、MDPI……），右下角出现悬浮胶囊；
3. 点「添加」直接收进阅读列表，或展开列表逐条操作（可选文件夹）；
4. 点工具栏插件图标可查看登录状态、当前页 PDF 列表和设置。

## ⚠️ 已知限制

- Chrome 内置 PDF 查看器页面无悬浮胶囊（角标与弹窗会提示并支持操作）；
- 依赖 Scholaread 网页端接口，官方改版后可能需要插件更新适配；
- `chrome://`、Chrome 应用商店等特殊页面无法使用。
